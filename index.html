WHAT IS GIT?

Introduction

Git is a popular version control system. It was created by Linus Torvalds in 2005, and has been maintained by Junio Hamano since then.

Why Use Git?

Tracking Changes: Git allows developers to track changes made to the codebase overtime.

Branching and Merging:Git enables developers to create branches to work on specific feature or fixes without affecting the main codebase.

Backup and Restore: Git provides a distributed system where every developers has a full copy of the repository

Collaboration: Git facilities collaboration among team members by allowing multiple developers to work on the same codebase simultaneously.

Code Reviews: Git supports code review processes by providing tools for reviewing chaneges.
